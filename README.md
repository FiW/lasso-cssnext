lasso-cssnext
==================

# Installation

Install the plugin:

```
npm install lasso-cssnext --save
```

Enable the plugin:

```javascript
require('lasso').configure({
    plugins: [
        {
            plugin: 'lasso-cssnext',
            config: {
                ... // See below for config options
            }
        },
        ...
    ]
});
```
